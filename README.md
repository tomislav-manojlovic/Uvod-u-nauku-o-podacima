# House Price Prediction - Ames Housing Dataset

Ovaj projekat koristi **Ames Housing Dataset** za predviđanje prodajne cene kuća (**SalePrice**) na osnovu njihovih karakteristika i uslova prodaje. Projekat je fokusiran na upoređivanje različitih modela mašinskog učenja kako bi se identifikovao onaj koji najtačnije predviđa cene.

## Dataset
- **Izvor:** [Ames Housing Dataset](https://www.kaggle.com/datasets/prevek18/ames-housing-dataset)
- **Opis:** Skup podataka sadrži više od 2.900 kuća i preko 80 atributa, uključujući fizičke, lokacijske i funkcionalne karakteristike.

## Cilj
- Razviti i uporediti modele regresije i napredne metode mašinskog učenja.
- Identifikovati faktore koji najviše utiču na cenu kuća.
- Odabrati model sa najboljom prediktivnom tačnošću.

## Tehnologije
- **Jezik:** R
- **Biblioteke:** tidyverse, caret, randomForest, xgboost, ggplot2
